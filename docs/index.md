# node simple static server
# usage
```js
import staticServer from 'path/to/index';
staticServer(<root>,<config>?);
```
## `root`
### the public file root path
## `config`
### you can config the hostname and port such as this
```json
{
  "host": "localhost",
  "port": 3000
}
```
